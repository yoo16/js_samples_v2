Event

jQueryのイベントハンドリング

Input / Click / Blur / Focus イベント

テキストボックスに 入力（input）、 フォーカス（focus）、 フォーカスアウト（blur）すると下のメッセージ欄に反映されます。 また「決定」ボタンをクリック（click）すると入力内容を確定できます。

Change イベント

プルダウンを変更すると右側の画像が切り替わります

Click イベント

リストを選択すると右側の画像が切り替わります

    Mouse イベント

    右側の画像にマウスをのせると画像が変わります。
    ボタンをクリックするとイベントがOFFになります。
